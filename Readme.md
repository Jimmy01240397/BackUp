# EasyFileService
A tcp file server.

## Usage

please install mono if you want to run in linux.
```
apt-get install mono-devel
```

### Server
use -h to get usage.
``` batch
.\EasyFileServiceClient help
```

start service
``` batch
.\EasyFileService.exe -p <port> <FileServiceRootPath>
```

### Client
use help to get usage.
``` batch
.\EasyFileServiceClient help
```
